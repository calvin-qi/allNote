### 2.算法的时间复杂度

```c++
//时间复杂度
void loveYou(int n){
	int i=1;
	while(i<=n){
		i++;
		printf("I love you %d\n,i");
	}//当n=3000时，循环3001次
	printf("I love you more than %d\n,n");
}

int main(){
	loveYou(3000);
}
```

3. 从小到大--->常--对--幂--指--阶
- 顺序执行的代码只会影响常数项，可以忽略（T(n)=3n+1003 = O(n) )
- 只需挑循环中的一个基本操作分析它的执行次数与n的关系即可
- 如果有多层循环嵌套，只需关注最深层循环循环了几次
