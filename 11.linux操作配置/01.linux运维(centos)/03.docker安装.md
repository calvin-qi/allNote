# docker安装



## 1、安装

```
# 1.用yum源安装查看是否已安装docker列表
yum list installed | grep docker

# 2.安装docker,-y表示不询问安装，知道安装成功，安装完后再次查看安装列表
yum -y install docker

# 3.启动docker
systemctl start docker

# 4.查看docker服务状态,看到绿色的字样active(running)表示已经安装成功
systemctl status docker

# 5.设置开机启动
systemctl enable docker.service
```



## 2、删除容器与镜像

```
# 1.docker ps -a查询已有的实例
docker ps -a

# 2.删除实例(容器ID)
docker stop dc57e734116c

# 3. 强制删除
docker rm -f dc57e734116c

# 4.删除镜像，先查看有哪些再删除
docker images
docker rmi -f 镜像id

# 5.删除所有的容器与镜像
 docker rm -f `docker ps -a -q` # 删除所有容器
 docker rmi -f `docker images -q` # 删除所有的镜像

```
